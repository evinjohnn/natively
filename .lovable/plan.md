

# Cluely.com Homepage Recreation

A pixel-perfect recreation of the Cluely.com landing page with full animations and interactive elements.

---

## 1. Navigation Bar
- Fixed top navbar with Cluely logo, navigation links (Pricing, Mobile, Blog), and a "Download" CTA button linking to the GitHub releases page
- Transparent on top, with blur/background transition on scroll
- Mobile hamburger menu

## 2. Hero Section — "#1 Undetectable AI for Meetings"
- Large gradient heading with animated text highlighting
- Subtitle text describing the product
- Primary CTA button linking to GitHub releases
- **Animated macOS-style mockup** with a dock bar (Safari, Zoom, Settings icons), a faux meeting UI with participant avatars, and an overlay showing the Cluely assistant panel — all built with CSS/HTML, with subtle entrance animations

## 3. "Four Ways We Make Your Meetings Better" Section
- Section heading with fade-in on scroll
- 2×2 grid of feature cards:
  - **AI answers questions for you** — with a mock chat UI card
  - **Instant follow-up emails** — with a mock email preview card
  - **Who are you really talking to?** — contact research card
  - **Beautiful meeting notes** — notes preview card
- Each card fades/slides in on scroll with hover lift effects

## 4. "Meeting Notes in 3 Steps" Section
- Centered heading with descriptive subtitle
- Three-step horizontal layout with numbered steps:
  1. Start Cluely
  2. End Cluely
  3. Get notes
- Each step shows a small mock UI illustration
- Staggered fade-in animations on scroll

## 5. "No Meeting Bots. 100% Undetectable" Section
- Bold centered headline with fade-in
- Subtitle/badge text
- Side-by-side comparison mockup: "Other AI Notetakers" (showing a visible bot) vs "Cluely" (undetectable, clean UI)
- Small Google Meet/Chrome extension icon badge
- Scroll-triggered entrance animation

## 6. Real-Time Transcription Section
- Left side: animated mock transcription UI with live-typing effect
- Right side: stats with large numbers and labels
  - **12+** Languages
  - **300ms** Response time
  - **95%** Transcription accuracy
- Numbers count up on scroll into view

## 7. FAQ Section
- "Frequently Asked Questions" heading
- Accordion-style expandable questions using Radix accordion component
- Smooth open/close animations
- Questions covering: What makes it different, who is it for, is it free, language support, enterprise support

## 8. Bottom CTA Section
- "Meeting AI that helps during the call, not after" heading
- "Try Cluely on your next meeting today" subtitle
- Green CTA button linking to GitHub releases

## 9. Footer
- Cluely logo and copyright
- Link columns: Company, Product, Legal
- Social media icon links
- Clean minimal layout

## Design & Animation Details
- **Color palette**: Soft blue/purple gradient hero background, white content sections, dark text
- **Typography**: Clean sans-serif, large bold headings, muted subtitles
- **Scroll animations**: Intersection Observer-based fade-in, slide-up, and scale-in for all sections
- **Hover effects**: Card lift with shadow, button color transitions, link underline animations
- **Counter animations**: Animated number counting for stats section
- **Smooth scrolling** throughout the page
